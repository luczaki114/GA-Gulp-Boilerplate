Learning gulp

  1. To get started with Gulp, I need a gulpfile.js
  2. In our gulpfile.js, we need to require a few libraries.
  3. We need gulp, gulp-uglify, gulp-concat, gulp-watch.
  4. Gulp is for running tastks
  5. Uglify is for minification.obfuscation
  6. Concat is for concatennation
  7. Watch will listen for changes to your code and auto-run the tasks when changes are made
  8. Now we install all our libraries and install with --save-dev
  9. After that, I create my first gulp.task named default.
  10. Gulp require that there is a default task.
